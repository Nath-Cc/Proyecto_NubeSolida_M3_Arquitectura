Clasificación de Servicios Cloud

IaaS Infraestructura como Servicio

  • Amazon EC2: Servidores virtuales escalables
  • Amazon VPC: Red privada virtual aislada
  • Servicios personalizados: Lógica de negocio custom

PaaS Plataforma como Servicio

  • Amazon RDS: Base de datos administrada
  • API Gateway: Gestión completa de APIs

FaaS Funciones como Servicio

  • AWS Lambda: Ejecución de código sin servidor
  • Escalado automático basado en eventos

SaaS Software como Servicio

  • Amazon Cognito: Autenticación de usuarios
  • Amazon SQS: Sistema de colas
  • Amazon SNS: Servicio de notificaciones
